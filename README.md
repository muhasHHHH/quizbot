\# QuizBot



\## Что это

Сайт-чат где можно пройти викторину. Бот задаёт вопросы,

ты отвечаешь, он считает очки. Всё сохраняется в базу данных.



\## Технологии

\- Python 3

\- Django 4.2

\- SQLite

\- HTML, CSS, JavaScript



\## Как запустить



pip install -r requirements.txt

python manage.py migrate

python manage.py fill\_questions

python manage.py runserver



Потом открыть: http://127.0.0.1:8000

\## Команды бота

\- привет — поздороваться

\- старт — начать викторину

\- python / математика / общие — выбрать категорию

\- A, B, C, D — ответить на вопрос

\- счёт — посмотреть результат

\- сброс — начать заново

\- помощь — список команд
